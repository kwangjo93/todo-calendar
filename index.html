    🚀 Google Calendar 연동 방법

    1. Google Cloud Console에 접속합니다.
      console.cloud.google.com
    2. 새 프로젝트를 생성하거나 기존 프로젝트를 선택합니다.
    3. API 및 서비스 → 라이브러리에서
      Google Calendar API를 활성화합니다.
    4. 사용자 인증 정보 → OAuth 2.0 클라이언트 ID 생성
      유형: 웹 애플리케이션
      승인된 원본:
    5. 발급받은 클라이언트 ID를 config.js 파일의
      CLIENT_ID 항목에 붙여넣습니다.
    6. 로컬 서버로 앱을 실행합니다:
      npx serve . 또는 python3 -m http.server 8080
